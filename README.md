# MobilityPlus

Enhanced movement equipment and mobility options for Techtonica.

## Features

### Equipment
- **Stilts MKII** - Increased height
- **Stilts MKIII** - Maximum height stilts
- **Speed Boots** - Increased movement speed
- **Jump Pack** - Enhanced jumping capability
- **Rail Runner MKII** - Improved grappling hook

### Placeable Items
- **Speed Pad** - Placeable speed boost zone
- **Jump Pad** - Placeable launch pad

## Requirements

- BepInEx 5.4.21+
- EquinoxsModUtils 6.1.3+
- EMUAdditions 2.0.0+
- TechtonicaFramework 1.0.0+

## Installation

Install via r2modman or manually place the DLL in your BepInEx/plugins folder.
