# Changelog

All notable changes to MobilityPlus will be documented in this file.

## [1.7.0] - 2026-01-07

### Changed
- All items now appear in unified "Modded" build menu tab
- Updated dependency on TechtonicaFramework 1.2.0
- Major version bump for vehicle and equipment improvements

### Fixed
- Build menu organization for cleaner UI

## [1.0.0] - 2025-01-05

### Added
- Initial release
- **Stilts MKII** - Increased height equipment
- **Stilts MKIII** - Maximum height stilts
- **Speed Boots** - Increased movement speed
- **Jump Pack** - Enhanced jumping capability
- **Rail Runner MKII** - Improved grappling hook
- **Speed Pad** - Placeable speed boost zone
- **Jump Pad** - Placeable launch pad
- Tech tree integration for all items
